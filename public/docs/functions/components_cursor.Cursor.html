    Function Cursor

    • Cursor(props: ArgsCursor): Element

      Renders a slider component that controls the opacity of selected Mapbox layers in real time.

      This component listens for user input on the range slider, updates its internal state, and applies the new opacity to all map layers whose IDs include the given substring(s).

      Parameters

      • props: ArgsCursor

        Object containing the component props.

        Arguments for the Cursor component.

        Represents the configuration options used to render a slider UI element that controls the opacity of specific map layers in a Mapbox map.

        • OptionalclassName?: string

          Optional CSS class name(s) for styling the slider container.

        • Optionaldef?: number

          Default slider value (from 0 to 100).

        • Optionalenabled?: boolean

          Whether dark mode is enabled.

        • include: string | string[]

          One or more substrings used to match map layer IDs. Layers containing any of these strings in their IDs will have their opacity adjusted by the slider.

        • map: RefObject<Map$1 | null>

          Reference to the Mapbox map instance.

        • name: string

          Display name shown next to the slider.

      Returns Element

      A JSX element representing the slider and its label.

      Example

      <Cursor
        name="Roads"
        include="road"
        map={mapRef}
        def={50}
        enabled={darkMode}
      />
