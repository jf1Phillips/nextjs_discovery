      Toggles real-time user geolocation tracking on a Mapbox map.

      When activated, this function continuously updates a marker to follow the user's current position using the browser's Geolocation API. When disabled, it stops tracking, removes the marker, and clears the active watch.

      Parameters

      • map: Map$1 | null

        The Mapbox map instance, or null if the map is not initialized.

      • marker: RefObject<Marker | null>

        A React ref holding the current Mapbox Marker used to display the user’s position.

      • loc: boolean

        A boolean flag indicating whether location tracking should be active (true) or disabled (false).

      • setLoc: Dispatch<SetStateAction<boolean>>

        A React state setter used to update the loc state when location availability changes.

      • watchId: RefObject<number | null>

        A React ref storing the ID of the active geolocation watch (if any), allowing cleanup or restart.

      Returns void

      Example

      // Enable geolocation tracking
      get_location(map, markerRef, true, setLoc, watchIdRef);

      Example

      // Disable geolocation tracking
      get_location(map, markerRef, false, setLoc, watchIdRef);

      Remarks

      • Uses navigator.geolocation.watchPosition() to continuously track position.
      • Ensures only one active watcher exists at a time (cleans up existing watchers before starting new ones).
      • Automatically removes the user marker and stops watching when loc is set to false.
      • Requests high-accuracy positioning with a 5-second timeout and zero cache age.
      • Safe-guards against missing browser geolocation support.
